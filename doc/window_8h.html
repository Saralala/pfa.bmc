When closing the window, checks if there is unsaved work.

When the window is requested to be closed, we need to check if there is unsaved work. We use this callback to prompt the user to save their work before they exit the application. With the "delete-event" signal, we can choose to cancel the close based on the value we return.

Parameters:
widgetThe window to close
event
editorThe BrailleMusicEditor structure containing flags for save purposes.
